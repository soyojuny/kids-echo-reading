# Kids Echo Reading

문서 기준(`plan.md`, `CLAUD.md`, `doc/*`)에 맞춰 구축하는 태블릿 중심 PWA 영어 따라읽기 프로젝트입니다.

## Quick Start

1. `.env.example`를 참고해 `.env.local` 작성
2. 의존성 설치: `npm install`
3. 개발 서버 실행: `npm run dev`
