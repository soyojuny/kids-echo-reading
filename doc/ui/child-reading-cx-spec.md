# Child Reading CX Spec

## Purpose
- 아동용 읽기 학습 UI의 기준 동작을 고정한다.
- 구현 전에 화면/상호작용/피드백 규칙을 합의한다.

## Scope
- 책 고르기, 읽기 연습, AI 추천 복습 화면
- 문장 단위 듣기/따라읽기 루프
- 단어 색상 누적 보상
- 페이지 단위 복습 및 최종 추천 복습

## Core UX
1. 아이는 책을 선택한다.
2. 읽기 화면에서 AI가 문장 전체를 읽는다.
3. 아이가 문장 전체를 따라 읽는다.
4. 맞게 발음한 단어만 파랑으로 누적 채움된다.
5. 남은 단어가 있으면 `"이 단어 한 번 더 해볼까?"`로 재시도한다.
6. 문장 완료 시 자동으로 다음 문장으로 이동한다.
7. 페이지 완료 시 남은 단어가 있으면 페이지 복습을 먼저 수행한다.
8. 마지막은 점수 페이지 대신 AI 추천 복습 페이지를 보여준다.

## Orientation Rule
- 가로(landscape): 기본 `2페이지 보기`
- 세로(portrait): 기본 `1페이지 보기`
- 방향 전환 시 기본값 자동 적용
- 사용자가 1/2페이지 토글을 직접 누르면 수동 선택 우선

## Feedback Rule
- 실시간 강조는 파랑 계열만 사용한다.
- 주황/빨강 진행 색상은 사용하지 않는다.
- 오답은 전체 문장 경고 대신 남은 단어 재시도 중심으로 안내한다.

## Invariants
- 읽기 루프 기본 단위는 문장이다.
- 보상은 누적형(맞은 단어 유지)이다.
- 결과 화면은 점수 중심이 아니라 학습 추천 중심이다.

## Acceptance Criteria
- 문장 단위 루프가 끊김 없이 동작한다.
- 맞은 단어만 누적 채움된다.
- 페이지 복습과 최종 추천 복습이 모두 동작한다.
- 가로/세로 기본 보기 정책이 적용된다.

## Out Of Scope
- 실제 STT/발음평가 엔진 상세 알고리즘
- 부모 리포트/통계 화면
